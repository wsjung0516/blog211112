Angular project 설계 고려사항

  1. TDD
    프로그램의 기본 개발은 항상 TDD를 염두에 두고 개발함. 테스트를 적용할 수 없는 함수 개발을 배제함. 프로그램 개발시에는 Angular 기본 test framework 인 jasmine의 기능을 보완하여, boilerplate 를 제거하여 test code를 단순하게 할 수 있는 library (spectator)를 적용함.
  2. State Management
    프로그램 전체의 각 모듈별로 변경되는 데이터의 상태를 모니터링하거나 업데이트하여 연관된 모듈들이 변경된 데이터를 즉시 수신이 가능하도록 함.
    적용기술: Redux ( ngxs : Angular전용 상태관리 프로그램) 
  3. Reactive Programming
    함수형 프로그램 방식으로 프로그램밍 하여 프로그램이 간결하고 오류 발생을 최소화할 뿐아니라, 생산성 및 유지보수성을 향상 시킴.
    적용기술: RxJS
  4. Error Handling
    프로그램 전체에서 발생하는 오류에 대해서 한개의 모듈에 처리함. Http오류, Server오류, 내부 프로그램 오류로 분류하여 표시하거나 서버에 저장함. 각 모듈에서는 오류처리를 별도로 하지 않고 단지 오류 발생한 정보를 전달하는 기능만 구현함. 
    적용기술: Angular에서 제공하는 ErrorHandler 모듈을 상속받아 적용함.
  5. Reactive Form Validation
    프로그램 전체의 UI에 대해서 입력되는 데이터의 Validation을 담당함. 각 프로그램에서는 Validation에 대한 기능을 별도로 구현하지 않고, 단지 요구되는 Validation을 등록하기만 하고, 모든 처리는 하나의 모듈에서 구현함. 
    적용기술: 필요한 validation 모듈은 개발하여 적용함.
  6. Http Interception
    서버와의 통신에서 발생하는 부가적인처리를 한 모듈 그룹에서 전담하도록 구현함. Http 통신 데이터의 Caching 기능, Http 통신시 Spinner 처리, 데이터 Converting JSON/JS, XML/JSON, Header정보를 추가함(  data type, contexts), Communication Status Message 전시, 물리적인 통신 오류 발생시 재전송  처리,  데이터 처리 시간등에 대한 로그처리, 테스트를 위한 Fake Server 기능, Authentication 기능 ( bearer token 추가, refresh token, redirect login page)  구현
    적용기술: Angular에서 제공하는HttpInterceptor 모듈을 상속받아 적용함.
  7. Overlay
    Angular Material, Bootstrap등이 제공하는 UI Component가 적합하지 않거나, 다이나믹하게 데이터를 전시하는 기능등을 위하여 별도 개발하여 구현함.
    적용기술: Angular Material, Angular CDK
  8. Data Table
    사용자의 요구사항에 맞게, 데이터 테이블에 다양한 기능을 포함하여야 하는 경우에 대비하여 필요 시 빠르게 수정 변경이 가능하게 구현함.
    적용기술: Angular Material, Angular CDK Material 
  9. Angular Animation
    특정 UI는 에니메이션 기능을 적용하여 사용자의 가독성 증가 및 데이터의 실시간성을 제공함.

Angular project design considerations

  1. TDD
    The basic development of the program is always developed with TDD in mind. Excludes the development of functions for which tests cannot be applied. When developing programs, used libraries (spectators) that can simplify test code and remove boilerplate, which complements the features of jasmine, Angular's native testing framework.
  2. State Management
    By monitoring or updating the status of the changed data for each module in the entire program, the related modules can receive the changed data immediately.
    Applied technology: Redux ( ngxs : State management program for Angular) 
  3. Reactive Programming
    By programming in a functional programming method, the program is concise and errors are minimized, and productivity and maintainability are improved.
    Applied technology: RxJS
  4. Error Handling
    Errors occurring throughout the program are handled in one module. Classified as Http error, Server error, and internal program error, displayed or stored in the server. Each module implements only the function of passing the error occurred information without separately handling the error.
    Applied technology: Inherit and apply the ErrorHandler module provided by Angular.
  5. Reactive Form Validation
    Responsible for validation of input data for the entire UI of the program. In each program, the function for validation is not implemented separately, only the required validation is registered, and all processing is implemented in one module.
    Applied technology: The necessary validation module is developed and applied.
  6. Http Interception
    It is implemented so that one module group is dedicated to the additional processing that occurs in communication with the server. Caching function of Http communication data, Spinner processing during Http communication, Data converting JSON/JS, XML/JSON, Communication Status Message display, retransmission in case of physical communication error processing, Implementation of log processing for data processing time, etc.
    Applied technology: Inherit and apply the HttpInterceptor module provided by Angular.
  7. Overlay
    If the UI component provided by Angular Material, Bootstrap, etc. is not suitable, or for the function of dynamically displaying data, it is separately developed and implemented.
    Applied technology: Angular Material, Angular CDK
  8. Data Table
    It is implemented so that it can be modified and changed quickly if necessary in case it is necessary to include various functions in the data table according to the user's requirements.
    Applied technology: Angular Material, Angular CDK Material 
  9. Angular Animation
    Certain UIs apply animation functions to increase user readability and provide real-time data.
